Git命令层级概述

Git命令分为上层友好命令(Porcelain)和底层基础命令(Plumbing)，构成完整的命令体系

上层命令 (Porcelain)

用户友好的高级接口

特点

易于使用和理解
提供一致用户体验
组合底层命令实现功能
处理常见错误和边界情况

常用命令

git clone
git commit
git branch
git checkout
git merge

底层命令 (Plumbing)

基础构建块命令

特点

专注于单一功能
不处理用户界面细节
可组合为复杂操作
适合脚本和自动化

核心命令

git cat-file
git hash-object
git update-index
git write-tree
git commit-tree
