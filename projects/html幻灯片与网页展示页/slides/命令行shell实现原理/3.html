命令提示符生成

命令提示符组成

用户名
当前登录用户
主机名
系统标识
当前目录
工作路径

实现流程

  1. 1
    获取环境变量 USER、HOSTNAME、PWD
  2. 2
    格式化字符串，生成提示符模板
  3. 3
    强制刷新输出缓冲区，确保立即显示
代码示例
# 获取环境变量
const char *GetUserName() {
    return getenv("USER");
}

# 生成命令提示符
void MakeCommandLine(char cmd_prompt[], int size) {
    snprintf(cmd_prompt, size, "[%s@%s %s]# ",
            GetUserName(), GetHostName(), GetPwd());
}
终端显示

user
