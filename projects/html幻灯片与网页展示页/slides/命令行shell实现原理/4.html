命令获取和解析

命令获取

安全读取用户输入
使用fgets()替代gets()防止缓冲区溢出
fgets(buffer, BUFFER_SIZE, stdin);

命令解析

分割命令行为单词(token)
使用strtok()函数进行字符串分割
// 首次调用
token = strtok(command, " ");

// 后续调用
while ((token = strtok(NULL, " ")))

引用处理

反斜线引用
转义单个字符
\$ → \$
单引号引用
所有字符字面化
'$USER' → $USER
双引号引用
部分字符扩展
"$USER" → user
解析示例
ls -l /home/user/Documents
ls
-l
/home/user/Documents
echo "Hello $USER, today is $(date)"
echo
"Hello user, today is 2023-06-15"

解析流程

  1. 1
    读取用户输入的命令行
  2. 2
    处理引用（单引号、双引号、反斜线）
  3. 3
    分割命令行为token（单词）
  4. 4
    执行各种扩展（变量、命令替换等）
  5. 5
    构建参数数组（argv格式）
  6. 6
    准备执行命令
