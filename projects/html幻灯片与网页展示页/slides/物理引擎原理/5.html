约束求解方法

物理引擎中解决碰撞和关节约束的核心算法

基于冲量的Sequential Impulse

逐个求解约束，模拟瞬时冲量作用

JkM-1JkTλk = -Jkv* - bk

算法步骤

  1. 计算所有物体的新速度 v*
  2. 对每个约束k，求解λk
  3. 应用冲量：vk+1 = vk* + M-1JkTλk
  4. 更新位置：xk+1 = xk + vk+1Δt

接触约束

防止物体互相穿透，保持物体分离

摩擦约束

限制切线方向相对运动

特点

实现简单
计算效率高
适合游戏物理引擎
稳定性较好

基于约束的雅可比矩阵法

使用拉格朗日乘数法求解约束系统

M•q̈ + JTλ = Fext

关键概念

  • 雅可比矩阵 J

    描述约束对状态变量的变化率

  • 拉格朗日乘数 λ

    表示约束力大小

等式约束

距离约束、旋转关节、滑轮等

不等式约束

接触约束、角度限制等

特点

理论完备
求解精度高
适用于复杂约束
计算成本较高

基于位置的动力学 (PBD)

核心思想

直接修正物体位置以满足约束，而非通过力或冲量

Δx = α(C(x) - x0)

α为松弛因子，C为目标约束

算法步骤

  1. 预测新位置
  2. 应用外力影响
  3. 求解约束
  4. 修正位置
  5. 更新速度

优缺点

实现简单
稳定性好
能量不守恒
不适合精确物理模拟
