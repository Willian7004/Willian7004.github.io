第四部分：Git的存储优化

存储优化概述

Git采用多种优化策略来减少存储空间占用，提高存储和传输效率

包文件(Pack File)

将多个对象压缩打包存储

  • 采用原始内容 + 增量内容形式存储
  • 自动压缩对象，减少磁盘占用
  • 优化网络传输效率
$ git gc # 触发自动打包
$ git verify-pack -v packfile.pack

索引文件(Index File)

快速定位包文件中的对象

  • 存储对象的大小、偏移、类型等元数据
  • 采用分层索引结构
  • 支持快速查找和重建文件快照

Delta存储技术

只存储对象之间的差异

  • 两种Delta存储类型：
  • OBJ_OFS_DELTA：基于文件偏移量的增量存储
  • OBJ_REF_DELTA：基于SHA-1的增量存储
  • 大文件只存储修改部分，大幅节省空间
